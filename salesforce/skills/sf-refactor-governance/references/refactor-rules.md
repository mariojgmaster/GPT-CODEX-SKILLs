# Refactor Rules

## Mandatory alignment question
- "Do you want to keep current legacy structure, or migrate to a modern pattern in this scope?"

## Allowed by default
1. Method extraction for readability.
2. Duplicate removal in scoped area.
3. Naming clarity improvements.
4. Performance-safe structural cleanup.

## Blocked by default
1. Cross-module architecture rewrites.
2. Framework migrations.
3. Behavior-changing refactors without approval.
