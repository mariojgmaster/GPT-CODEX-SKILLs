# Discovery Checklist

1. Identify feature owner path or package.
2. Identify touched artifact categories:
   - Apex classes/triggers/tests
   - LWC/Aura artifacts
   - Flow and metadata
   - Permissions and labels
3. Map dependencies:
   - runtime
   - deployment
   - data model
4. Define minimal validation set:
   - targeted tests
   - dry-run scope
5. Register out-of-scope artifacts.
