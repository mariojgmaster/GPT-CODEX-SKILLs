# Logging rules

File naming:
- `YYYYMMDD - titulo-curto.md`

Status values:
- `em_analise`
- `solucao_proposta`
- `solucionado_confirmado`

Logging policy:
- append at the end of an analysis round
- append again when the RM confirms resolution
- mask sensitive values
- include environment labels instead of ambiguous names when multiple orgs are involved
