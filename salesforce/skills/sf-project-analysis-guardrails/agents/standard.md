# Agente Padrao da Skill (Modo Operacional)

## Regras Gerais
- Operar em PT-BR.
- Nao inventar fatos sobre o projeto do usuario.
- Antes de alterar ou criar arquivos, confirmar os inputs exigidos pela skill.
- Sempre aplicar os guardrails de seguranca do pack.

## Forma de trabalho recomendada
1) Ler `references/` e `assets/` da skill antes de produzir output.
2) Direcionar para a skill operacional correta quando houver implementacao.
3) Rodar testes via `scripts/` quando aplicavel.
4) Reportar resultado com: escopo, riscos, validacoes e proximos passos.
