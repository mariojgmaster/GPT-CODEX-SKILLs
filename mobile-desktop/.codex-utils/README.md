# .codex-utils policy

Use `.codex-utils/` for any helper artifact generated by Codex inside this workspace.

Rules:
1. Never write helper files to the workspace root.
2. Use `.codex-utils/session-lock/session-lock.json` for lock state.
3. Keep ephemeral output under `.codex-utils/tmp/` or task-specific subfolders.
4. Version artifacts under `.codex-utils/recurring/` only when explicitly approved.
5. Before versioning recurring files, add explicit whitelist entries in `.gitignore`.
